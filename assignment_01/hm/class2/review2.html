ORB SPACE

https://www.orb-space.com

DESIGN

Cute design, minimalistic with a single horizontal movement feature, no scrolling down.

USABILITY

A but hard to understand, the info bar is represented by a red dot, no intuitive instinct to view it as a potential information bar. You kind of want to scroll down, but can’t so you feel a bit “constricted” in your movement. Using the website on a smartphone is a better experience as we are programmed to just swipe instinctively.

CONTENT

All projects are shown directly on the main page, which is good but you feel like that is the only thing they’re presenting except for their description on top.

CREATIVITY

Orignal, but simple. Could have a bit more movement or features to make it even more interesting. The way pictures are showing up when we select a project is a nice touch to make it more interactive.
